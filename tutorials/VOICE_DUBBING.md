# 配音制作教程

## 配音文件规范

| 项目 | 要求 |
|------|------|
| 格式 | MP3（兼容最好）或 FLAC（无损） |
| 采样率 | 44100Hz |
| 声道 | 单声道（游戏配音） |
| 命名 | `角色名/台词内容.mp3` |
| 目录 | `game/vocal/角色名/` |

## 完整制作流程

### 方法一：AI 语音合成（高速）

```
第 1 步：整理台词
   → 从剧本中提取所有角色台词
   → 按角色分文件整理

第 2 步：选择 TTS 工具
   → 在线：火山引擎 TTS、讯飞、Azure
   → 本地：VITS、GPT-SoVITS

第 3 步：逐条生成
   → 一次生成一句台词
   → 检查发音和语气
   → 不满意的重新生成

第 4 步：后期处理（Audacity）
   → 降噪（Noise Reduction）
   → 统一音量（Normalize）
   → 裁剪首尾空白（Truncate Silence）

第 5 步：导出命名
   → 格式：MP3 320kbps
   → 命名：完整的台词文字.mp3（方便识别）
   
第 6 步：放入游戏
   → 放到 game/vocal/对应角色目录/
   → 在剧本中用 -语音文件.mp3 关联
```

### 方法二：真人录制（高质量）

```
第 1 步：找配音演员
   → 朋友/配音社团/配音平台
   
第 2 步：录制
   → 安静环境，电容麦克风
   → 一人分饰多角录多遍
   
第 3 步：后期
   → 同 AI 方案的 Audacity 流程
```

## 剧本中关联配音

```webgal
; 语法：角色名:"台词" -语音文件.mp3;
陈泽:"今天天气真好。" -陈泽/今天天气真好。.mp3;

; 不带语音
陈泽:"这是一句没有语音的台词。";

; 旁白没有语音
:这是一段旁白。;
```

## 批量转换脚本

本仓库提供了 FLAC → MP3 的批量转换脚本：

- `tools/scripts/convert_flac_to_mp3.bat`（Windows）
- `tools/scripts/convert_flac_to_mp3.py`（Python）

## 注意事项

- AI 语音合成目前不会有肖像权问题（仅音频）
- 长句子建议分段生成，语气更自然
- 关键情绪台词（哭、笑、怒吼）建议真人录制
- 语音文件大小尽量控制在 200KB 以内
- 文件名不要包含特殊字符 `\/:*?"<>|`
