# 视频生成教程

## 视频在游戏中的作用

WebGAL 支持在剧本中插入 MP4 视频作为过场动画，用于：
- 开场动画（展示标题/氛围）
- 关键剧情演出（摔手机、表白等名场面）
- 结局动画（HE/NE/BE 分歧演出）

## 视频规格

| 参数 | 要求 |
|------|------|
| 分辨率 | 1920x1080（16:9） |
| 格式 | MP4（H.264 编码） |
| 时长 | 5-30 秒 |
| 码率 | 8-15 Mbps |
| 帧率 | 24/30 fps |

## 剧本中调用视频

```webgal
; 基础播放
playVideo:开场动画.mp4;

; 带音量控制
playVideo:过场.mp4 -volume=100;

; 不可跳过（适合关键剧情）
playVideo:表白.mp4 -skipOff;

; 音量 50%，不可跳过
playVideo:结局.mp4 -volume=50 -skipOff;
```

## 生成方案对比

### 方案 1：ComfyUI + AnimateDiff（推荐）

功能最强，完全本地运行，可精确控制。

- **教程**：B站 [啦啦啦的小黄瓜](https://space.bilibili.com/219572544)
- **适合**：角色动态、精细动画
- **工作流**：Prompt → ControlNet (姿势) → AnimateDiff → 输出 MP4

### 方案 2：Banana

在线 API，无需部署。

- **适合**：简单场景动画、快速出片
- **操作**：上传图片 → 选择模型 → 生成

### 方案 3：即梦

字节跳动出品，部分功能有人脸限制。

- **适合**：无人物场景、风景动画
- **注意**：目前人脸相关的生成有限制

## 视频后期处理

```
生成原始视频
    ↓
剪映/Premiere 剪辑
    ↓ 裁剪到 1920x1080
    ↓ 调色统一风格
    ↓ 添加字幕/特效
    ↓ 导出 H.264 MP4
放入 game/video/
```
