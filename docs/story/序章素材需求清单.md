## 七、素材制作建议

### AI 绘画工具推荐
- **立绘**：Stable Diffusion + ControlNet（保持角色一致性）
- **场景**：Midjourney / Stable Diffusion
- **道具**：可直接 AI 生成或实拍转绘

### 配音建议
- 可使用 AI 语音工具（如 Azure TTS、ElevenLabs）
- 或找真人配音（推荐 B 站/配音社群）
- 陈泽少年期需要正太/少年音
- 王女开需要少女音
- 王展虎需要青年音

### 音效来源
- 免费音效库：freesound.org
- 商用音效：Epidemic Sound、AudioJungle
- 自制：手机录音 + 后期处理

## 八、检查清单

制作完成后请确认：

- [ ] 所有立绘背景透明（PNG 格式）
- [ ] 立绘尺寸统一（建议 800x1200 或更高）
- [ ] 场景图尺寸匹配游戏分辨率（建议 1920x1080）
- [ ] 音频格式为 MP3， bitrate 128kbps 以上
- [ ] 文件命名与代码引用一致
- [ ] 所有 P0 素材已完成
- [ ] 主要台词语音已录制
