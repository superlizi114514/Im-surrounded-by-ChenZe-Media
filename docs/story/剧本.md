# 《完蛋！我被陈泽传媒包围了》官方宣传剧本

> 一部关于青春、梦想与选择的互动式恋爱视觉小说

---

## 📖 故事简介

**你叫王妍**，14 岁，黑龙江绥棱县的一名普通初二学生。

2012 年夏天，你路过一个少年的人生低谷——

他叫**陈泽**，父母离异，跟着奶奶生活。因为偷了奶奶的钱去网吧，被你撞见。

从那天起，两个孤独的灵魂开始交织。

**QQ 头像上那个"妍"字**，**路灯下笨拙的表白**，**运动会台阶上递来的零食**……

但青春从来不只有甜蜜。

母亲的反对、现实的残酷、梦想的重量……

**你的一次选择，将决定两个人的命运。**

当然你可以选择攻略其他传媒的人，以不同的视角，不同的人

---



## 📜 完整剧情线路

### 【序章】网吧偷钱事件（2012 年夏）

**时长**：8-10 分钟

**核心剧情**：
- 陈泽与奶奶吵架，摔碎翻盖手机
- 你路过他家楼下，目睹他冲出门
- 网吧撞见他偷钱买水
- QQ 头像变成"妍"字
- KTV 邀约被拒（妈妈接的电话）
- **路灯下表白**："处对象不？"
- 运动会见面，矛盾激化
- **关键选择**：捡起书包 or 扔进垃圾桶

**名场面台词**：
> "求你了，别告诉我奶。"
> "'妍'...是你名字里的字。我换这个头像，是因为...我喜欢你。"
>

---

### 【A 线】甜宠线：没扔书包（一直在一起）

#### A1 辍学线：直播创业

**入口条件**：第一章第二幕选择"跟我去干直播吧"

**核心剧情**：
- 母亲发现恋情，赶你出门
- 你选择跟陈泽一起生活
- 住进"小臭屋"，开始直播
- 最难的时候，陈泽为 Q 币给老铁磕头
- 黑白加入，三人组成立
- 芝士酸奶（抠脚大汉）送 500 皮肤号
- 老崔刷 18.8 皇冠结识
- 阴阳玉佩传说开启

**名场面**：
```
【陈泽】妍，我不会让你后悔的。
【王妍】我相信你。

【旁白】最难的时候，为了几个 Q 币，陈泽也只能给直播间的老铁磕头。
【旁白】我坐在床边，看着他的背影，眼泪止不住地流。
```

**结局预设**：
- 陈泽线：从校服到婚纱，戒指刻"1103"
- 黑白线：从 0 培养感情，3011 伏笔回收
- 事业线：三人组继续，专注直播事业

---

#### A2 校园线：久别重逢

**入口条件**：第一章第二幕选择"再等等吧"

**核心剧情**：
- 母亲威胁让陈泽转学
- 你被迫提分手，走廊诀别
- 最后一局井字棋
- 陈泽去大城市，你继续学业
- 高中三年，每天看他直播
- 大学改名，进入陈泽传媒实习
- **随机 PK 重逢**，两人都选择沉默
- 委婉拒绝后，交集变少

**名场面**：
```
【旁白】屏幕闪烁了几下，然后出现了一个熟悉的面孔。
【旁白】是陈泽。
【旁白】他也愣住了。
【陈泽】......你好，开开。
【王妍】......你好，泽哥。
【旁白】我们都选择了沉默。
```

**三种结局**：
| 结局 | 触发条件 | 核心台词 |
|------|---------|---------|
| BE：擦肩而过 | 选择"后悔了" | "有些人，一旦错过，就是一辈子" |
| NE：各自安好 | 选择"不后悔" | "有些感情，不一定要拥有" |
| OE：来日方长 | 选择"不知道" | "有些故事，还没有结束" |

---

### 【B 线】分手线：扔了书包

#### B 线共同剧情段（初三全年）

**核心剧情**：
- 开学座位表：展虎和你同桌，陈泽被调角落
- 展虎的追求：早餐、接送、陪伴
- 陈泽的角落：默默关注，眼神交汇
- 矛盾与挣扎：在两人之间摇摆

**毕业分岔口**：
```
展虎："王妍，我喜欢你，从初一开始就喜欢。"
展虎："和我在一起吧，我们一起去外地上学。"

选择：
- "我愿意和你试试" → B1 甜宠线
- "对不起，我不能" → B2 分手线
- "我想转学"（隐藏） → B3 恨海情天
```

---

#### B1 甜宠线：展虎与王妍

**核心剧情**：
- 高中情侣，"爱丽丝"爱称由来
- 陈泽的疏远，三人关系微妙
- 毕业后展虎做直播，你成为助播
- 陈泽传媒合作邀约，重逢
- 展虎的不安，最终选择

**结局预设**：
- 展虎单一线：从路灯下到婚纱
- 陈泽线：1103 分的等待
- 海王线（BE）：一无所有

---

#### B2 分手线：巨婴的直播日记

**核心剧情**：
- 拒绝展虎，独自开启直播
- 起名"巨婴"，自嘲风格
- 深夜刷到展虎视频，内心波动
- 陈泽传媒面试，重逢陈泽
- 展虎发来消息："那里...不适合你"
- 直播间连麦 PK 选择

**结局预设**：
- 陈泽线（主线 True End）：角落到舞台
- 事业线（独立 End）：巨婴女王
- 复合展虎（Open End）：未完待续

**名场面**：
```
【旁白】毕业后的第三个月，我还是没有找到工作。
【旁白】决定试试直播。
【旁白】注册账号时，需要起一个网名。
【旁白】我想起了和展虎在一起的日子。
【王妍】就叫...巨婴吧。
```

---

#### B3 恨海情天（隐藏 BL 线）

**触发条件**（需同时满足）：
- 展虎好感度 ≥ 38
- 陈泽好感度 ≤ 10
- 毕业分岔口选择"我想转学"

**核心剧情**：
- 王妍转学，不想再看到两人争吵
- 陈泽与展虎关系升温
- 从敌对到和解，产生别样情愫
- 酒后电话，模糊的告白
- 禁忌之恋，社会压力

**四种结局**：
| 结局 | 核心台词 |
|------|---------|
| 恨海情天（BE） | "两个人最后为了对方都能好过，分开了" |
| 朋友以上（NE） | "他们还是朋友。最好的朋友。也只能是朋友" |
| 禁忌之恋（TE） | "不在乎世人的眼光。只要能和你在一起" |
| 未说完的话（HE） | "话没说完，电话挂断了。已经回不去了" |
