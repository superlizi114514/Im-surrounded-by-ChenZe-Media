## 二、详细场景设计

### 事件 1：和奶奶吵架摔翻盖机（家里）

**场景**：陈泽家里，晚上

```
┌─────────────────────────────────────────────────────┐
│  起因：奶奶不让陈泽玩手机                            │
├─────────────────────────────────────────────────────┤
│  - 陈泽想用奶奶的翻盖机玩游戏/打电话                │
│  - 奶奶不给，说"学生玩什么手机"                     │
│  - 陈泽叛逆期，跟奶奶吵起来                         │
│  - 一怒之下摔了翻盖机（电池飞出）                   │
│  - 奶奶生气了："你摔！有本事你别回来！"             │
│  - 陈泽摔门而出，离家出走                           │
└─────────────────────────────────────────────────────┘
```

**剧本片段**：

```renpy
label childhood_event1_fight:
    scene bg home_living_room_night with fade

    n "那天晚上，陈泽和奶奶吵了一架。"
    n "因为一部手机，一部老式翻盖机。"

    show grandma angry at right
    show chenze_teen angry at left

    grandma "学生玩什么手机！不准玩！"

    chenze_teen "我就玩一下怎么了！"

    grandma "说了不准就是不准！"

    n "陈泽一气之下，抓起手机就往地上摔。"

    play sound phone_smash

    n "啪的一声，翻盖机电池盖飞出，电池掉在地上。"

    grandma "你...你摔！有本事你别回这个家！"

    n "陈泽愣了愣，转身冲出了家门。"
    n "身后传来奶奶的喊声——"

    grandma "走了就别回来！"

    hide all
    scene bg street_night with fade

    n "那是初二的一个晚上，天气很闷。"
    n "陈泽漫无目的地走在街上，不知道去哪。"

    return
```

### 事件 2：楼下偶遇王女开（离家出走后）

**场景**：陈泽家楼下/小区花园，晚上

```
┌─────────────────────────────────────────────────────┐
│  偶遇：王女开刚好路过                                │
├─────────────────────────────────────────────────────┤
│  - 王女开看到陈泽坐在楼下花坛边                    │
│  - 头发乱糟糟的，眼睛红红的                        │
│  - 旁边放着摔坏的翻盖机（缠着胶布）                │
│  - 王女开问"怎么了"                                │
│  - 陈泽沉默，然后简单说了几句                      │
│  - 王女开安慰他                                    │
│  - 陈泽说"我没事，你先回去吧"                      │
│  - 王女开走后，陈泽一个人坐到很晚                  │
└─────────────────────────────────────────────────────┘
```

**剧本片段**：

```renpy
label childhood_event2_meetup:

    n "王女开路过小区花园时，看到了一个人影。"

    show chenze_teen sad at center

    n "他坐在花坛边上，头发乱糟糟的。"
    n "旁边放着一部缠着胶布的翻盖机。"

    chenze_teen "..."
    chenze_teen "没事。"

    n "王女开走过去，在他旁边坐下。"

    e "你这是...跟你奶吵架了？"

    chenze_teen "..."
    chenze_teen "嗯。"

    n "陈泽捡起那部摔坏的手机，翻了翻。"

    chenze_teen "手机也摔坏了..."
    chenze_teen "这是我奶的，现在好了，我也没家了。"

    menu:
        "我...":
            "你别这么说":
                e "你别这么说，你奶肯定是气话。"
                chenze_teen "气话？她都说让我别回来了。"
                $ add_chenze_love(3)

            "我陪你坐会":
                e "我陪你坐会吧。"
                chenze_teen "..."
                chenze_teen "你不回去啊？"
                e "没事，反正也不急。"
                $ add_chenze_love(5)

            "手机能修吗":
                e "手机能修吗？"
                chenze_teen "修啥啊，都摔成这样了。"
                chenze_teen "先用胶布缠着，能开机就行。"
                $ add_chenze_love(2)

    n "两人坐了一会，谁也没说话。"
    n "夏天的夜晚，知了在树上叫个不停。"

    chenze_teen "你...你先回去吧。"
    chenze_teen "我一个人待会。"

    e "那你..."

    chenze_teen "我没事，真的。"
    chenze_teen "就是...想一个人静一静。"

    n "王女开看了看他，起身走了。"

    hide chenze_teen

    n "陈泽一个人坐到很晚，直到小区的路灯都熄了。"
    n "那部缠着胶布的手机，他一直带在身边。"

    return
```

### 事件 3：偷钱去红蚂蚁网吧（被撞见）

**场景**：红蚂蚁网吧，下午

```
┌─────────────────────────────────────────────────────┐
│  起因：陈泽没钱上网，偷拿奶奶的钱                   │
├─────────────────────────────────────────────────────┤
│  - 陈泽想去网吧打英雄联盟，但没钱                  │
│  - 从奶奶抽屉偷拿了 100 块钱                          │
│  - 跑到红蚂蚁网吧（当地最好，2 元/小时）              │
│  - 开机打游戏，选 ADC                               │
│  - 王女开路过后进去（借充电宝/路过）                │
│  - 看到陈泽，旁边钱包露出红钞票                    │
│  - 陈泽发现王女开，手抖，英雄被秒杀                │
│  - 陈泽慌乱，但这次没摔手机（因为已经摔过了）      │
│  - "求你了，别告诉我奶！"                          │
└─────────────────────────────────────────────────────┘
```

**剧本片段**：

```renpy
label childhood_event3_internet_cafe:

    n "而他旁边放着的钱包里，露出了一张红色的钞票。"

    e (内心) "那是...陈泽？"
    e (内心) "他怎么在这？不对，那是他奶的钱吧？"

    n "就在这时，陈泽余光扫到了门口的她。"

    chenze_teen "卧槽！"

        "我...":
            "我看到了":

    chenze_teen "你不会告诉她吧？"

    menu:
        "我...":
            "不会说的":
                e "我不会说的。"
                chenze_teen "真的？！"
                e "真的。"
                chenze_teen "..."
                chenze_teen "你...你人还挺好的。"
                $ add_chenze_love(8)

            "以后别这样了":
                e "以后别这样了。"
                chenze_teen "嗯，我知道了。"
                chenze_teen "谢谢你。"
                $ add_chenze_love(5)

    n "最后陈泽还是回家了，游戏也没打完。"
    n "他走之前，回头看了王女开一眼。"

    chenze_teen "今天的事...谢谢你。"

    return
```
