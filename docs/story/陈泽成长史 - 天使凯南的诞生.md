### 事件 5：陈泽沉迷红蚂蚁网吧，苦练天使凯南

**时间**：初二下学期（表白事件后）

**地点**：红蚂蚁网吧

**背景**：
- 陈泽表白后，虽然王女开没有明确接受，但两人关系更近了
- 陈泽开始沉迷英雄联盟，尤其是天使凯南这个英雄
- 他几乎每天都在红蚂蚁网吧，一坐就是一整天

**剧本片段**：

```renpy
label childhood_event5_grind:
    scene bg internet_cafe with fade

    n "表白之后，陈泽好像变了个人。"
    n "他还是天天去红蚂蚁网吧，但不再是为了消磨时间。"

    show chenze_teen focused at center

    n "他开始苦练一个英雄——天使凯南。"
    n "那时候的凯南，上单强势，能抗能打。"

    n "陈泽每天都练，从早到晚。"
    n "补刀、连招、timing、支援...每一个细节都不放过。"

    show message 网吧老板："小陈，你又来了啊"
    show message 网吧老板："今天还是老位置？"

    chenze_teen "嗯，老位置，给我留着。"

    n "他坐在最里面的位置，屏幕上是天使凯南的图标。"
    n "一局接着一局，输了就再来，赢了也继续。"

    # 蒙太奇
    scene bg internet_cafe_montage with fade

    n "春天，他在练凯南的对线细节。"
    n "夏天，他在练凯南的团战 timing。"
    n "秋天，他在练凯南的支援路线。"
    n "冬天，他已经是这条街有名的'凯南绝活哥'。"

    show message 网吧路人："卧槽，这凯南太猛了吧！"
    show message 网吧路人："这谁啊？"
    show message 网吧老网瘾："他啊，天天来，就知道练凯南。"
    show message 网吧老网瘾："说是...要打上职业。"

    return
```
