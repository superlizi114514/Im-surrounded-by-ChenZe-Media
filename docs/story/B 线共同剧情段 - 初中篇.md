# B 线总设计：初中共同剧情段

## 线路总览

```
序章（初二暑假）
│
├─ 选择陈泽 → A 线（校园线/辍学线）
│
└─ 选择分手 → B 线统一入口
    │
    └─ B 线共同剧情段（初三→初中毕业）
        │
        ├─ 第一幕：开学座位表
        ├─ 第二幕：展虎的追求
        ├─ 第三幕：陈泽的角落
        ├─ 第四幕：矛盾与挣扎
        │
        └─ 毕业分岔口（★分支点）
            ├─ 接受展虎 → B1 甜宠线
            ├─ 拒绝展虎 → B2 分手线
            └─ 【隐藏】转学事件 → B3 恨海情天
```

## B 线共同剧情段详细设计

### 前置条件

```
进入 B 线共同剧情段的条件：
- 序章第八幕选择"把包扔进垃圾桶"
- 或者序章第八幕选择"跟展虎走"
```

### 第一幕：开学座位表（初三上学期）

**场景**：初三教室，开学第一天

**核心事件**：
- 王艳华老师宣布座位表
- 展虎和王妍被安排为同桌
- 原本坐这个位置的陈泽被调到角落

```
王艳华："王展虎，你坐王妍旁边。"

陈泽默默收拾书包，走向教室角落。
他没有看我们，只是低着头。

王妍（内心）："那个曾经为我出头的男孩，
就这样被他最讨厌的人，挤到了角落里。"
```

**选项设计**：
```
Choose:
    "看向陈泽":          # 陈泽好感 +3，展虎好感 -2
    "接受这个安排":      # 展虎好感 +3
    "假装无所谓":        # 无变化
```

---

### 第二幕：展虎的追求（初三上→下）

**场景**：教室/学校/校外

**核心事件**：
- 展虎开始追求王妍
- 每天带早餐、送零食、接送上下学
- 王妍内心挣扎

**关键场景**：

#### 场景 1：早餐

```
展虎："给，热乎的。"

他递过来一个纸袋，是校门口那家的煎饼果子。
加蛋加肠，是我喜欢的搭配。

王妍（内心）："他怎么会知道我的喜好？"
```

**选项**：
```
Choose:
    "收下":              # 展虎好感 +5
    "谢谢，不用了":      # 展虎好感 -3，陈泽好感 +2
    "我自己有":          # 无变化
```

#### 场景 2：放学路上

```
展虎："我送你回家吧。"

他推着自行车，走在我旁边。
夕阳把他的影子拉得很长。

王妍（内心）："这样的日子，持续了多久呢？"
```

**选项**：
```
Choose:
    "让他送":            # 展虎好感 +5
    "不用了，我想一个人走走":  # 展虎好感 -2
    "嗯，走吧":          # 展虎好感 +3
```

---

### 第三幕：陈泽的角落（初三全年）

**场景**：教室角落

**核心事件**：
- 陈泽在角落里默默关注王妍
- 偶尔的眼神交汇
- 王妍内心的挣扎

**关键场景**：

#### 场景 1：眼神交汇

```
数学课上，我回头看了一眼角落。
陈泽正好抬头，我们的目光对视了一秒。

他先移开了视线，低头继续做题。

王妍（内心）："他...是不是还在意我？"
```

**选项**：
```
Choose:
    "对他笑一下":        # 陈泽好感 +5，展虎好感 -3
    "迅速转回头":        # 无变化
    "写纸条递给他":      # 陈泽好感 +8，展虎好感 -5
```

#### 场景 2：偶遇

```
放学后，我在车棚遇到了陈泽。
他一个人在那里，推着自行车发呆。

陈泽："...好巧。"

王妍（内心）："是真的巧，还是他故意的？"
```

**选项**：
```
Choose:
    "好巧，一起走吗":    # 陈泽好感 +8，进入陈泽暗线
    "嗯，我先走了":      # 陈泽好感 -3
    "...":              # 无变化
```

---

### 第四幕：矛盾与挣扎（初三下学期）

**场景**：教室/操场/校外

**核心事件**：
- 展虎的攻势越来越明显
- 陈泽的沉默让王妍心疼
- 王妍在两人之间摇摆

**关键场景**：

#### 场景 1：展虎的告白前奏

```
展虎："王妍，毕业之后...你有什么打算？"

他看着我，眼神里有我看不懂的情绪。

王妍（内心）："他...想说什么？"
```

**选项**：
```
Choose:
    "还没想好":          # 展虎好感 +2
    "可能去外地上学":    # 展虎好感 -3
    "你呢":              # 展虎好感 +3
```

#### 场景 2：陈泽的沉默

```
那天放学，陈泽在楼下等我。

陈泽："你...和展虎...在一起了？"

王妍（内心）："他是在乎的吗？"
```

**选项**：
```
Choose:
    "没有":             # 陈泽好感 +5
    "和你有什么关系":   # 陈泽好感 -10，进入决裂线
    "嗯":               # 陈泽好感 -5，展虎好感 +5
```

## 毕业分岔口（★分支点）

### 场景：毕业典礼后

**地点**：学校操场
**时间**：初三毕业当天下午

```
展虎："王妍，我有话跟你说。"

他把我叫到操场，背对着教学楼。
阳光很好，蝉鸣很响。

展虎："我喜欢你，从初一开始就喜欢。"
展虎："和我在一起吧，我们一起去外地上学。"
```

### 分支选项

```
Choose:
    "我愿意和你试试":      # → B1 甜宠线
    "对不起，我不能":      # → B2 分手线
    "让我再想想":          # → 进入 B3 隐藏判定
```

---

## B3 隐藏触发条件

### 触发要求（需同时满足）

| 条件 | 要求 |
|------|------|
| 展虎好感度 | ≥ 60 |
| 陈泽好感度 | ≤ 20 |
| 关键选项 | 选择"让他送"≥3 次 |
| 隐藏事件 | 触发"陈泽角落偶遇"≥2 次 |

### 触发后的额外选项

```
如果满足 B3 触发条件，在毕业分岔口会出现隐藏选项：

Choose:
    "我想转学":          # → B3 恨海情天线
    "我愿意和你试试":    # → B1 甜宠线
    "对不起，我不能":    # → B2 分手线
```
