## 一、英雄设定修正

| 项目 | 设定 |
|------|------|
| **英雄** | 正义天使·凯尔（Kayle） |
| **定位** | 上单/ADC，后期大核，16 级无敌 |
| **特点** | 前期弱势，后期成神，需要发育 |
| **象征** | 陈泽的成长——从叛逆少年到可靠男人 |

### 为什么是凯尔？

```
凯尔的英雄特质：
├─ 1-5 级：弱势，需要发育 → 陈泽叛逆期，不懂事
├─ 6-10 级：开始有战斗力 → 陈泽开始努力，想变强
├─ 11-15 级：强势期 → 陈泽小有名气，"凯尔绝活哥"
├─ 16 级：完全体，无敌 → 陈泽成熟，能保护想保护的人
│
└─ 象征意义："我等你长大，也等你回来"
```

| 细节 | 描述 |
|------|------|
| **被动** | 登峰造境（16 级完全体）—— 象征陈泽的坚持 |
| **Q 技能** | 耀焰审判——"我的喜欢，像火焰一样炽热" |
| **W 技能** | 星体恩典（加血加移速）——"我想治愈你，保护你" |
| **E 技能** | 星剑合一（被动伤害）——"每次看到你，心跳都加速" |
| **R 技能** | 凝滞（无敌 2.5 秒）——"即使世界崩塌，我也为你抵挡" |
