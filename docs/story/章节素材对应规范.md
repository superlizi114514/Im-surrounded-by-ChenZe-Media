# 《完蛋！我被陈泽传媒包围了》章节素材对应规范

## 一、素材目录结构

```

├── 序 人物素材/           # 独立序章专用素材（童年回忆 - 网吧偷钱事件）
├── 第一章 人物素材/        # 第一章专用素材（办公室重逢）
├── 第二章 人物素材/        # 第二章专用素材（暧昧升级）

├── 第五章 人物素材/        # 第五章专用素材（陈泽线 - 危机考验）
├── 第六章 人物素材/        # 第六章专用素材（陈泽线 - 求婚婚礼）
```

## 五、代码规范

### 5.1 剧本文件头部注释

每个剧本文件必须在头部声明所属章节和可用素材：

```renpy
# ============================================
# 独立序章：网吧偷钱事件（童年回忆）
# 时期：初中时期（2015-2016）
# 可用素材目录：照片/序 人物素材/
# 禁止使用：成年陈泽立绘、办公室背景、咖啡店背景
# ============================================
```

### 5.2 素材使用检查清单

在提交代码前，必须确认：
- [ ] 本剧本使用的立绘都在对应章节素材目录中
- [ ] 本剧本使用的背景都在对应章节素材目录中
- [ ] 没有使用其他章节的专属素材
- [ ] 通用素材（bg_black、UI、音效、BGM）可以使用

## 六、违规示例与修正

### 违规示例 ❌

```renpy
# 独立序章（童年回忆）中使用了成年陈泽立绘
label childhood:
    show chenze normal at center  # ❌ 错误！童年回忆只能用少年陈泽
```

### 修正方案 ✅

```renpy
# 独立序章（童年回忆）中使用少年陈泽立绘
label childhood:
    show chenze_teen_normal at center  # ✅ 正确！
```

## 七、素材命名规范

### 立绘命名

```
{角色}_{时期}_{表情}.png

示例:
- chenze_teen_normal.png    # 少年陈泽普通表情
- chenze_adult_happy.png    # 成年陈泽开心表情
- nvkai_teen_sad.png        # 少女女主难过表情
```

### 背景命名

```
bg_{地点}_{时间/状态}.png

示例:
- bg_home_living_room_night.png   # 家里客厅晚上
- bg_office_morning.png           # 办公室早上
- bg_street_night.png             # 街道晚上
```

## 八、执行与监督

1. **每次提交前**：运行素材检查脚本
2. **Code Review 时**：检查素材使用是否符合规定
3. **发现违规**：立即修正，并更新本文档
