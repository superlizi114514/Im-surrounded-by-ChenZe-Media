# 陈泽童年回忆：网吧偷钱事件

> 版本：v1.0
> 核心：**调皮少年 × 暗恋女神 × 被抓包 × 表白**
> 时间线：初中时期（序章之前传/回忆杀）

---

## 一、剧情框架

```
偷钱去网吧
    │
    ▼
女主路过撞见
    │
    ▼
陈泽慌乱摔手机
    │
    ▼
楼下偶遇 简短对话
    │
    ▼
女主发现他头像变成"妍"字
    │
    ▼
女主发"？？？"
    │
    ▼
陈泽表白
    │
    ▼
结局分支（接受/拒绝）
```
