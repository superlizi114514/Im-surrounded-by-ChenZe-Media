# B 线路总入口与分支设计

## 线路总览

```
序章（初二暑假）
│
├─ 选择陈泽（好感度≥10 捡包） → A 线（校园线/辍学线）
│
└─ 选择分手（扔包/跟展虎走） → B 线统一入口
    │
    └─ B 线共同剧情段（初三→初中毕业）
        │
        ├─ 第一幕：开学座位表
        ├─ 第二幕：展虎的追求
        ├─ 第三幕：陈泽的角落
        ├─ 第四幕：矛盾与挣扎
        │
        └─ 毕业分岔口（★分支点）
            │
            ├─ 展虎好感≥50，选择"我愿意" → B1 甜宠线
            ├─ 选择"对不起，我不能" → B2 分手线
            └─ 【隐藏】展虎≥60+ 陈泽≤20，选择"我想转学" → B3 恨海情天
```

## 序章入口：第八幕分支

**场景**：小区垃圾桶旁，夜晚

**前置条件**：完成序章第 1-7 幕

### 选项设计

```
陈泽站在原地，手里拿着那个 Nike 背包。
他看了看我，又看了看旁边的展虎。

陈泽："王女开，你什么意思？"

Choose:
    "无视展虎，走到陈泽身边":
        # 条件：陈泽好感度≥10 才能触发
        $ sweetheart_route = True
        $ chenze_fav += 10
        $ zhanhu_fav -= 5
        → 进入 A 线（校园线/辍学线）

    "把包掠过来，丢到垃圾桶里":
        $ breakup_route = True
        $ chenze_fav -= 20
        $ zhanhu_fav += 10
        → 进入 B 线统一入口

    "看着展虎，轻声说'送我回去吧'":
        # 【隐藏选项】需要展虎好感≥15
        if zhanhu_fav >= 15:
            $ zhanhu_fav += 15
            $ chenze_fav -= 30
            $ breakup_route = True
            → 进入 B 线统一入口（展虎好感加成）
        else:
            # 好感不足，自动切换到"扔包"选项
            → 进入 B 线统一入口
```

## 毕业分岔口（★分支点）

**场景**：学校操场，毕业当天下午

```
展虎："王妍，我喜欢你，从初一开始就喜欢。"
展虎："和我在一起吧，我们一起去外地上学。"

# 检查 B3 隐藏条件
if zhanhu_fav >= 38 and chenze_fav <= 10:
    # 满足 B3 触发条件，显示隐藏选项
    Choose:
        "我愿意和你试试":
            $ b1_route = True
            → 进入 B1 甜宠线

        "对不起，我不能":
            $ b2_route = True
            → 进入 B2 分手线

        "我想转学":
            $ b3_route = True
            → 进入 B3 恨海情天线

elif zhanhu_fav >= 30:
    # 满足 B1 条件
    Choose:
        "我愿意和你试试":
            $ b1_route = True
            → 进入 B1 甜宠线

        "对不起，我不能":
            $ b2_route = True
            → 进入 B2 分手线

    # 展虎好感不足，只能进入 B2

            $ b2_route = True
```

### 毕业分岔口阈值

| 线路 | 展虎好感度 | 陈泽好感度 |

| B1 | ≥ 30 | 不限 |

## 线路流程图

```
序章（初二暑假）
│
├─ 捡包（好感≥10） → A 线
│
└─ 扔包/跟展虎走 → B 线
    │
    └─ 共同剧情段（初三）
        │
        ├─ 座位表事件
        │   ├─ 看向陈泽 → 陈泽 +3，展虎 -2
        │   ├─ 接受安排 → 展虎 +3
        │   └─ 无所谓 → 无变化
        │
        ├─ 早餐事件（多次）
        │   ├─ 收下 → 展虎 +5
        │   ├─ 拒绝 → 陈泽 +2，展虎 -3
        │   └─ 我自己有 → 无变化
        │
        ├─ 放学事件（多次）
        │   ├─ 让他送 → 展虎 +5
        │   ├─ 拒绝 → 展虎 -2
        │   └─ 嗯走吧 → 展虎 +3
        │
        ├─ 眼神事件（多次）
        │   ├─ 笑一下 → 陈泽 +5，展虎 -3
        │   ├─ 转回头 → 无变化
        │   └─ 写纸条 → 陈泽 +8，展虎 -5
        │
        ├─ 偶遇事件（多次）
        │   ├─ 一起走 → 陈泽 +8，展虎 -5
        │   ├─ 先走了 → 陈泽 -3
        │   └─ ... → 无变化
        │
        └─ 毕业分岔口
            │
            ├─ 展虎≥60 + 陈泽≤20 → 【隐藏】我想转学 → B3
            │
            ├─ 展虎≥50 → 我愿意 → B1
            │
            └─ 其他 → 对不起/再想想 → B2
```
