### 命名规范

```
背景：bg_场景名_时间段.png
  例：bg_downstairs_night.png

立绘：char_角色名_表情.png
  例：char_chenze_teen_nervous.png

CG：cg_章节号_事件名.png
  例：cg_prologue_confession.png

道具：prop_道具名.png
  例：prop_flipphone_broken.png
```

### 透明背景处理流程

1. AI 生成时用**纯白色背景**
2. 用 Photoshop 一键选择白色区域删除
3. 导出为 PNG 格式保留透明通道
4. 立绘和道具图统一按此流程处理

## 七、批量生成技巧

### 保持风格一致

```
- 同一批次使用相同 seed
- 固定同一个模型/LoRA（推荐 Realistic Vision 或 FLUX.1）
- 光照方向保持一致
- 使用相同的色彩分级预设
```

### 立绘表情批量（真人）

```
1. 先生成基础表情（中性）
2. 使用同一人物 LoRA/参考图
3. 用 inpaint 功能只改脸部表情
4. 保持发型服装不变
5. 使用 Roop/ReActor 等换脸工具保持一致性
```

### 道具抠图技巧

```
1. 生成时要求纯白背景
2. Photoshop 用"选择主体"或"色彩范围"
3. 删除白色背景
4. 添加轻微内阴影增加立体感
```

### 真人风格特别提示

```
- 添加 film grain 增加电影感
- 使用 color grading 统一色调
- 添加 light leaks 增加怀旧感
- 使用 LUTs 进行后期调色
```
