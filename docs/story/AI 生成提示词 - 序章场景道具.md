## 六、批量生成建议

### 场景批量生成

```
使用同一 seed，调整时间/天气参数：
- 白天场景：daytime, sunlight, clear sky
- 夜晚场景：night, street lamp, dark blue sky
- 黄昏场景：sunset, orange sky, warm lighting
```

### 立绘批量生成

```
使用同一 seed，只调整表情参数：
- 基础立绘生成后，用 inpaint 修改表情
- 保持头发、服装一致
```

### 道具批量生成

```
白色背景单独生成后，用 Photoshop 合成到场景中
- 道具图生成时加上简单阴影
- 保持光照方向一致
```

## 八、技术规范

### 输出格式

| 类型 | 格式 | 分辨率 | 用途 |
|------|------|--------|------|
| 背景 | PNG | 1920x1080 | Ren'Py 场景背景 |
| 立绘 | PNG (透明底) | 720x1280 | Ren'Py 角色立绘 |
| CG | PNG | 1920x1080 | 事件插画 |
| 道具 | PNG (透明底) | 512x512 | 场景合成素材 |
| UI | PNG (透明底) | 1920x1080 | 界面元素 |

### 命名规范

```
背景：bg_场景名_时间段.png (例：bg_downstairs_night.png)
立绘：char_角色名_表情.png (例：char_chenze_teen_nervous.png)
CG：cg_章节号_事件名.png (例：cg_prologue_confession.png)
道具：prop_道具名.png (例：prop_flipphone_broken.png)
UI：ui_元素名.png (例：ui_choice_box.png)
```

**文档结束**

生成建议：
1. 优先完成🔴最高优先级资源
2. 同一批次的图使用相同 seed 保持风格一致
3. 立绘表情用 inpaint 批量修改效率最高
4. 道具图生成后统一抠图做透明背景
