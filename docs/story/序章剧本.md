# 序章：网吧偷钱（2012-2013，初二夏天）

**时长**：8-10 分钟
**字数**：1500-2000 字
**主角**：王女开（王妍）
**视角**：女主视角为主，部分场景切换陈泽视角
**登场角色**：陈泽、王女开（王妍）、陈泽奶奶（声音/短信）、精神小伙（路人）、妈妈（声音）

---

## 时间线总览

```
家里摔手机（翻盖机电池飞出）→ 冲下楼偶遇（手里没手机）→ 偷钱去网吧（买水被撞见）→
楼下偶遇（偷钱当晚）→ QQ 头像变"妍" → KTV 邀约 → 路灯下表白 → 运动会见面 → 分支选项
```
