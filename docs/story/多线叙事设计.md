# 多线叙事设计 - 女粉向剧情优化

## 核心概念

**"如果当时我拒绝了陈泽"**

在第三章表白/求婚后，增加**拒绝线**，形成双线叙事：

```
第三章抉择点
    │
    ├─ 接受线（现有 HE/NE 线）
    │   └─► 第 4-6 章 → 三种结局
    │
    └─ 拒绝线（新增）
        └─► 以粉丝视角看直播 → 听他说起"那个人"
```

---

## 拒绝线剧情框架

### 核心体验

玩家以**普通粉丝/观众**的身份，在直播间听陈泽提起"有个青梅竹马拒绝了我也"，产生：
- 遗憾感
- 好奇心（那个女生是谁？）
- 代入感（我就是那个女生）

### 剧情流程

```
拒绝线入口（第三章选择"我们做朋友吧"）
    │
    ▼
第 4 章：疏远期
- 两人关系尴尬
- 陈泽刻意保持距离
- 女主逃避式离职
    │
    ▼
第 5 章：直播间重逢
- 女主在家看直播
- 陈泽在直播中提起"有个很重要的人"
- 弹幕炸了："泽哥有情况！"
    │
    ▼
第 6 章：再次抉择
- 陈泽私下联系女主
- "我还想再试一次"
- 玩家再次选择：接受/继续拒绝
    │
    ├─ 再次拒绝 → BE：错过
    └─ 接受 → HE：兜兜转转还是你
```

## 双线对比

| 维度 | 接受线 | 拒绝线 |
|------|--------|--------|
| 情感基调 | 甜蜜推进 | 遗憾→反转 |
| 玩家心理 | "这就是我的 HE" | "我错过了什么"→"还能挽回吗" |
| 核心场景 | 求婚、婚礼 | 直播间告白、再次相遇 |
| 结局数量 | 3 个 (HE/NE/BE) | 2 个 (HE/BE) |
| 解锁条件 | 第三章选择接受 | 第三章选择拒绝 |

## 女粉代入感设计

### 1. 拒绝线的心理活动

```
"我为什么要拒绝？"
- 害怕失去朋友关系
- 觉得自己配不上他
- 需要时间想清楚

这些都是女生在现实中会有的犹豫。
```

### 2. 直播间的"公开告白"

```
让所有观众/粉丝见证：
- "泽哥认真的"
- "那个女生是谁啊"
- "好深情呜呜呜"

这种"被公开表白"的感觉，女粉最吃。
```

### 3. 兜兜转转还是你

```
拒绝后他没有放弃
等待后他还在原地
这种"非你不可"的执念，最能打动人。
```

|--------|------|------|
| 🔴 高 | 拒绝线第 5 章（直播间告白） | 核心情感爆点 |

## 文案金句库

### 可用于直播间告白场景

1. "我有个从小一起长大的人，她可能在看直播。"
2. "如果你在看的话，我想跟你说一句：我等你。"
3. "被拒绝也没关系，我习惯了。从小到大，都是她说了算。"
4. "有人说我条件好，不至于这样。但他们不懂，有些人就是一辈子只能遇到一个。"

### 可用于再次相遇场景

1. "我拒绝的不是你，是那个还没准备好的自己。"
2. "兜兜转转这么久，原来你一直在原地。"
3. "这次换我走向你，好不好？"

## 后续扩展

### 其他角色线的拒绝线

同样的结构可以套用到黑白、老崔、刘军线：

```
黑白线拒绝 → 听朋友说起"他好像喜欢你"
老崔线拒绝 → 直播间听他开玩笑提起"有个姑娘"
刘军线拒绝 → 通过别人转达"他说理解你的选择"
```

### 隐藏彩蛋线

```
如果四条线都拒绝 → 触发"独美"结局
"事业比恋爱重要，我选择搞事业！"
```
