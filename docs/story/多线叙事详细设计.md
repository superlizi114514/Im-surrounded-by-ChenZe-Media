## 二、核心分歧点设计（第三章）

### 第三章剧情节点

**场景**：公司团建度假村，傍晚 sunset

**触发条件**：第一章、第二章好感度累积

**关键对话**：

```renpy
label chapter3_common:
    scene bg resort_sunset with fade

    chenze "[PLAYER_NAME]，有件事我想跟你说。"
    chenze "我喜欢你，从小时候就开始了。"

    # 这时其他人也来了
    show heibai normal at right
    show laocui normal at left
    show liujun normal at center

    n "四个人的目光同时聚焦在我身上。"
    n "我必须要做出选择了..."

    menu:
        "我的回答是...":
            # 选项 A：接受
            "我也喜欢你，阿泽":
                $ add_chenze_love(20)
                jump chenze_route_chapter4  # 进入接受线

            # 选项 B：拒绝（新增）
            "我们...先做朋友吧":
                $ add_chenze_love(-5)
                jump reject_route_chapter4  # 进入拒绝线

            # 选项 C：犹豫
            "我不知道...给我点时间":
                $ add_chenze_love(-3)
                # 默认进入拒绝线分支
                jump reject_route_chapter4
```

## 三、接受线详细设计（已有框架优化）

### 接受线·第四章：青梅竹马的羁绊

**核心情感**：甜蜜、正式公开、回忆杀

**场景流程**：

| 场景 | 内容 | 情感点 |
|------|------|--------|
| 办公室早晨 | 陈泽喊"女朋友" | 甜蜜称呼 |
| 直播间公开 | 带女主上直播，介绍"这是我女朋友" | 公开承认 |
| 回忆杀 | 小时候一起玩过家家 | 青梅竹马专属 |
| 深夜谈心 | "这么多年还能在一起，不容易" | 感慨珍惜 |

**关键对话**：

```renpy

show message 弹幕：「这就是青梅竹马吗，慕了慕了！」

    "我...":
        "打招呼":
            e "大家好。"
            chenze "哈哈，听见没，这是我女朋友！"
        "躲在后面":
            e "躲在后面。"
            chenze "别害羞啊，以后要经常见的。"
```

**回忆杀 CG 建议**：

## 四、拒绝线详细设计（新增）

### 拒绝线·第四章：疏远与逃避

**核心情感**：尴尬、疏离、后悔、逃避

**场景流程**：

| 场景 | 内容 | 情感点 |
|------|------|--------|
| 办公室第二天 | 陈泽刻意保持距离 | "我们还是朋友" |
| 疏远蒙太奇 | 不再带早餐、不再送回家 | 失去后才珍惜 |
| 女主内心独白 | "我为什么要拒绝？" | 后悔情绪 |
| 离职决定 | "我想请几天假" | 逃避式离开 |

**关键对话**：

- 女主已离职在家
- 晚上习惯性打开陈泽直播间
- 这是她第一次以"观众"身份看他直播
