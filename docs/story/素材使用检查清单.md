## 提交前检查清单

### 1. 头部注释检查

- [ ] 剧本文件头部是否有素材声明注释？
- [ ] 注释是否包含以下信息：
  - 章节名称
  - 时期设定（童年/成年）
  - 可用素材列表
  - 通用素材说明

**示例**：
```renpy
# ============================================
# 第一章：暧昧升级（成年时期）
# 时期：2026 年成年时期
# 可用素材：成年陈泽立绘、黑白立绘、老崔立绘、刘军立绘
# 可用背景：办公室系列、街道系列
# 通用素材：bg_black、UI、音效、BGM
# ============================================
```

## 发现问题时的处理流程

1. **发现违规素材使用**
   - 立即修正剧本文件
   - 重新运行验证脚本
   - 确认修正成功

2. **发现规范文档缺失**
   - 更新 `docs/章节素材对应规范.md`
   - 更新 `verify_assets.py` 配置
   - 通知团队成员

3. **发现验证脚本误报**
   - 检查关键词配置
   - 更新 `CHARA_KEYWORDS` 或 `BG_KEYWORDS`
   - 重新测试验证脚本
