# 陈泽童年回忆剧本优化设计

> 版本：v1.0
> 目的：增强青梅竹马情感厚度，让女粉更有代入感
> 核心：**具体细节 > 模糊概括**

---

## 一、现有问题分析

### 原剧本问题

```renpy
# 序章中的回忆
n "那时候我们还小，一起上学，一起玩游戏..."
n "后来他做了主播，我们之间的联系渐渐少了。"

# 第 4 章中的回忆
chenze "你还记得小时候咱俩一起玩的事吗？"
e "当然记得。"
chenze "那时候你总跟在我屁股后面跑。"

# 第 6 章中的回忆
chenze "你还记得小时候咱俩玩过家家吗？"
chenze "那时候你说长大了要嫁给我。"
```

**问题**：
1. 太模糊——"一起玩"具体玩什么？
2. 缺少画面感——没有具体场景、对话、情感
3. 缺少互动——都是陈泽在说，女主没有参与感
4. 缺少独家记忆——"过家家"太通用，不是他们专属

---

## 二、优化方向

### 核心设计理念

| 维度 | 优化前 | 优化后 |
|------|--------|--------|
| 场景 | 模糊概括 | 具体场景（时间、地点、人物） |
| 对话 | 旁白叙述 | 角色对话（有来有回） |
| 情感 | "他对我好" | "他只对我好" |
| 细节 | 无 | 专属信物/口头禅/动作 |

### 童年回忆三大核心场景

```
场景 1：放学路上的糖（日常温暖）
   │
   ├─ 地点：小学到家的路上
   ├─ 物品：五毛钱一包的糖
   └─ 情感：他把好吃的都留给你
   │
   ▼
场景 2：被欺负了他出头（保护欲）
   │
   ├─ 地点：学校操场
   ├─ 冲突：男生抢你橡皮
   └─ 情感：他永远护着你
   │
   ▼
场景 3：过家家约定（感情线）
   │
   ├─ 地点：他家院子里
   ├─ 信物：拉钩上吊一百年不许变
   └─ 情感：小时候的玩笑他当真了
```

## 三、详细剧本设计

### 场景 1：放学路上的糖

**插入位置**：序章咖啡店场景，女主回忆部分

```renpy
# ============================================
# 优化后：序章回忆杀 - 放学路上的糖
# ============================================

label prologue_childhood_memory:
    scene bg school_gate with fade

    n "那时候我们上小学，每天一起放学回家。"
    n "陈泽家在我家隔壁，走路要经过一个小卖部。"

    show chenze_child happy at center

    n "小陈泽每天都会掏出五毛钱，买一包糖。"

    chenze (小) "给，你最喜欢的草莓味。"

    e (小) "你不吃吗？"

    chenze (小) "我不爱吃甜的，你吃吧。"

    n "后来我才知道，他是舍不得吃，都留给我。"

    # 转场回到现实
    scene bg cafe_interior with fade
    hide chenze_child

    n "思绪被拉回现实，陈泽把点好的咖啡推到我面前。"
    n "拿铁，不加糖。"

    show chenze happy at center

    chenze "还是老样子，对吧。"

    menu:
        "你还记得...":
            e "你还记得我爱喝什么。"
            chenze "那当然。"
            chenze "你喜欢草莓味的糖，不爱喝苦咖啡。"
            chenze "睡觉要抱枕头，不吃香菜..."
            e "停停停，别说了！"
            chenze "怎么，我说错了？"
            n "我看着他，突然觉得这些年，他好像一直没变。"
            $ add_chenze_love(10)

        "这么多年了":
            e "这么多年了，你还记得。"
            chenze "怎么可能忘。"
            n "他的眼神很认真，认真到我有些不敢看。"
            $ add_chenze_love(5)
```

**女粉代入点**：
- "他把好吃的都留给你"——被偏爱的感觉
- "他只对你好"——专属感
- "这么多年他还记得"——被放在心上的感觉

### 场景 2：被欺负了他出头

**插入位置**：第 1 章或第 2 章，陈泽吃醋后女主回忆

```renpy
# ============================================
# 新增：第 2 章回忆杀 - 他永远护着你
# ============================================

label chapter2_childhood_memory_protect:
    scene bg school_playground with fade

    n "小学三年级，有个男生总欺负我。"
    n "抢我的橡皮，扯我的辫子。"

    show boy_mean at center

    n "那天他又来抢我的文具盒，我急得快要哭了。"

    boy_mean "就不给你，你能怎么样！"

    n "然后陈泽出现了。"

    show chenze_child angry at center

    chenze (小) "你干嘛欺负她！"

    boy_mean "关你什么事！"

    chenze (小) "她是我妹妹，你欺负她就是欺负我！"

    n "陈泽把我护在身后，和那个男生打了一架。"
    n "最后他鼻青脸肿地赢了。"

    # 回家路上
    scene bg street_afternoon with fade

    show chenze_child hurt at center

    e (小) "疼不疼..."

    chenze (小) "不疼。"
    chenze (小) "以后他再欺负你，你就告诉我。"
    chenze (小) "我保护你。"

    n "那是我第一次明白，有个人会永远护着你。"

    # 转场回到现实
    scene bg office_workspace with fade
    hide chenze_child

    n "回过神来，陈泽正站在我面前。"
    n "他因为我和黑白闹得不愉快，像极了小时候那个为我出头的少年。"

    show chenze serious at center

    chenze "你...跟他们很熟吗？"

    menu:
        "还行吧":
            e "还行吧。"
            n "陈泽皱了皱眉，表情有些复杂。"
            chenze "你别跟他们走太近。"
            chenze "这些人...不靠谱。"
            n "我突然想起小时候他说的话——'我保护你'。"
            n "这么多年过去了，他还是想保护我。"
            $ add_chenze_love(8)
```

**女粉代入点**：
- "他为了你和别人打架"——保护欲爆棚
- "她是我妹妹，你欺负她就是欺负我"——占有欲
- "以后他再欺负你，你就告诉我"——永远的后盾

### 场景 3：过家家约定（核心情感爆点）

**插入位置**：第 6 章求婚场景，替换原有单薄的过家家对话

```renpy
# ============================================
# 优化后：第 6 章过家家回忆 - 完整版
# ============================================

label chenze_route_chapter6_childhood_full:
    # 求婚前的回忆闪回
    scene bg courtyard_sunset with fade

    n "记忆突然回到了那个夏天。"
    n "我们大概七八岁，在他家院子里玩过家家。"

    show chenze_child happy at center
    show player_child happy at left

    chenze (小) "今天你当新娘，我当新郎。"

    e (小) "为什么我是新娘？"

    chenze (小) "因为你漂亮啊。"

    n "小陈泽从地上摘了一朵野花，笨拙地插在我头上。"

    e (小) "那...新郎新娘要说什么？"

    chenze (小) "要说...要说一辈子在一起。"

    n "两个小孩伸出小拇指，拉钩上吊。"

    chenze (小) "拉钩上吊一百年不许变！"
    chenze (小) "变了就是小狗！"

    e (小) "那说好了，长大了你娶我。"

    chenze (小) "嗯！我娶你！"

    n "那时候的我们，以为长大了是很遥远的事。"
    n "以为说过的话，都会实现。"

    # 转场回到现实
    scene bg restaurant with fade
    hide chenze_child
    hide player_child

    show chenze serious at center

    n "现实中的陈泽，单膝跪地，打开戒指盒。"

    chenze "[PLAYER_NAME]，你还记得小时候咱俩玩过家家吗？"

    menu:
        "记得...":
            e "当然记得。"
            chenze "那时候你说长大了要嫁给我。"
            chenze "我等这句话，等了很多年。"
            $ add_chenze_love(10)

        "那都是小时候的事了":
            e "那都是小时候的事了。"
            chenze "但我可一直记着呢。"
            chenze "你说的每一句话，我都记着。"
            $ add_chenze_love(5)

        "你怎么突然提这个":
            e "你怎么突然提这个？"
            chenze "因为我想告诉你——"
            chenze "小时候的玩笑，我当真了。"
            $ add_chenze_love(8)

    chenze "小时候你说要嫁给我，我可一直记着呢。"
    chenze "今天，我想正式问你一个问题。"

    n "陈泽单膝跪地，打开了盒子。"
    n "里面是一枚闪闪发光的戒指。"

    chenze "[PLAYER_NAME]，你愿意嫁给我吗？"
    chenze "这一次，不是过家家。"
    chenze "是真的一辈子。"
```

**女粉代入点**：
- "因为你说过的话，我都记得"——被重视的感觉
- "小时候的玩笑，我当真了"——执念
- "这一次，不是过家家，是真的一辈子"——从童话到现实的浪漫

## 四、新增专属信物系统

### 信物：草莓味糖果

**设计理念**：每个青梅竹马故事都需要一个专属信物，让观众看到这个物品就想起他们。

```renpy
# 信物出现场景汇总

# 场景 1：序章咖啡店
chenze "给，你最喜欢的草莓味。"
n "他从口袋里掏出一包草莓味糖果，递给我。"
n "这么多年了，他还记得我喜欢的口味。"

# 场景 2：第 4 章甜蜜互动
chenze "张嘴。"
e "干嘛？"
chenze "给你糖。"
n "他把一颗草莓味糖果塞进我嘴里。"
chenze "甜吗？"
e "甜。"
chenze "那就好，你甜在心里，我在你心里。"

# 场景 3：第 6 章婚礼
n "婚礼上，每个宾客都收到了一份小礼物——"
n "一包草莓味糖果，和一张卡片。"
n "卡片上写着：'谢谢你见证我们的爱情'"
n "陈泽说，这是我们的专属味道。"
```

## 五、新增专属口头禅/动作

### 口头禅

| 台词 | 使用场景 | 情感含义 |
|------|----------|----------|
| "从小到大，你还能跑了？" | 女主想逃时 | 占有欲 + 宠溺 |
| "我还不了解你？" | 女主嘴硬时 | 熟悉 + 亲密 |
| "你屁股一撅我就知道..." | 调侃女主 | 青梅竹马专属 |
| "有事找我，随时都在" | 女主需要帮助时 | 承诺 + 守护 |

### 专属动作

| 动作 | 使用场景 | 情感含义 |
|------|----------|----------|
| 摸头杀 | 安慰女主时 | 宠溺 + 保护 |
| 拉手腕 | 着急时 | 占有欲 + 急切 |
| 剥糖纸 | 给女主吃糖时 | 细心 + 照顾 |
| 挡在身前 | 有危险时 | 保护欲 |
