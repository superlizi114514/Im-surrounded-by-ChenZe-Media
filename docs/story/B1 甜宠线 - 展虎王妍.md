## 线路概述

**入口条件**：
- 完成序章（选择分手）
- 完成 B 线共同剧情段（初三）
- 毕业分岔口选择"我愿意和你试试"
- 展虎好感度 ≥ 30

**主线**：王妍与展虎从高中情侣到直播搭档的甜宠路线
**时间线**：初三毕业→高中→大学/直播→2026 年

**场景**：高一教室，开学第一天

**核心事件**：
- 王艳华老师再次安排展虎和王妍同桌
- 原本坐这个位置的陈泽被调到更远的角落
- 展虎的占有欲开始显现

```
王艳华："王展虎，你还是坐王妍旁边吧。"

陈泽默默收拾书包，从第一排调到了最后一排角落。
他经过我身边时，停了一秒，又继续走了。

王妍（内心）："这是第几次了？因为他，陈泽一直在角落里。"
```

**选项**：
```
Choose:

    "专注于新课本":      # 展虎好感 +3
    "假装没看见":        # 无变化
```

**核心事件**：

- 原本娇小的王妍慢慢变圆润

```

王妍："别这样叫我..."

展虎："不知道为什么，我很喜欢这个称呼。"
展虎："以后你就是我的爱丽丝，好不好？"
```

**选项**：
```
Choose:

    "假装生气":          # 展虎好感 +5（傲娇）
    "不太喜欢":          # 展虎好感 -3
```
