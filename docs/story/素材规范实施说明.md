# 章节素材规范实施说明

> 版本：v1.0
> 实施日期：2026-03-24

---

## 一、核心规定

**相应章节只能用相应的人物素材**

这意味着：
1. 童年回忆章节只能使用少年时期的立绘和背景
2. 成年时期章节只能使用成年时期的立绘和背景
3. 陈泽线专属章节只能使用陈泽线允许的素材

## 三、素材目录结构

```
照片/
├── 序 人物素材/           # 独立序章专用（童年回忆）
│   ├── 小泽 *.png         # 少年陈泽立绘
│   └── 网吧.png 等背景    # 童年回忆背景
├── 陈泽/                  # 成年陈泽立绘
├── 陈泽成长史素材/         # 成长史专用素材
└── 通用素材/              # 通用素材（所有章节可用）
```

## 四、剧本头部注释规范

每个剧本文件必须在头部添加素材声明注释：

```renpy
# ============================================
# 序章：重逢（成年时期）
# 时期：2026 年成年时期
# 可用素材：成年陈泽立绘、咖啡店背景、餐厅背景
# 通用素材：bg_black、UI、音效、BGM
# ============================================
```

## 六、违规示例与修正

### 违规示例 ❌

在童年回忆章节使用成年陈泽立绘：

```renpy
label childhood:
    scene bg_internet_cafe
    show chenze normal at center  # ❌ 错误！应使用少年陈泽
```

### 修正方案 ✅

```renpy
label childhood:
    scene bg_internet_cafe
    show chenze_teen_normal at center  # ✅ 正确！
```

## 八、常见问题

### Q: 通用素材有哪些？

A: 以下素材可以在所有章节使用：
- `bg_black` - 纯黑背景（转场/结束）
- `bg_gradient_*` - 渐变背景（UI 界面）
- `ui_*` - UI 界面元素
- `sfx_*` - 音效文件
- `bgm_*` - BGM 音乐文件

### Q: 如何添加新的素材规范？

A:
1. 在 `docs/章节素材对应规范.md` 中添加新章节的素材规定
2. 在 `verify_assets.py` 的 `CHAPTER_CONFIG` 中添加配置
3. 在剧本文件中添加头部注释

### Q: 验证脚本报错怎么办？

A:
1. 查看报错信息，确定是哪个文件违规
2. 检查违规的素材类型（立绘或背景）
3. 修改剧本，使用符合规定的素材
4. 重新运行验证脚本
