# B3 触发结局：恨海情天

## 线路概述

**入口条件**（隐藏线，需同时满足）：
- 完成序章（选择分手）
- 完成 B 线共同剧情段（初三）
- 展虎好感度 ≥ 38（理论最大值，需要全选展虎选项）
- 陈泽好感度 ≤ 10
- 毕业分岔口选择"我想转学"（隐藏选项）

**主线**：王妍转学后，陈泽与展虎关系升温→触发特殊结局线
**核心冲突**：三角关系→王妍退出→两人情愫→禁忌之恋
**时间线**：初三毕业→高中→成年后
**关键词**：BL 线、兄弟情、禁忌之恋、三败俱伤

```

他帮我打饭，我帮他记笔记。
这算什么？

陈泽（内心）："没有她的教室，原来也可以这样平静。"
```

## 结局预设

### 结局一：恨海情天（Bad End）

```
后来是展虎优先喜欢上了陈泽。
陈泽一直以来也只是把他当做一个哥哥。

虎表达心意的时候，陈泽一直在吃惊和犹豫。
但是最后还是一试了。

两个人在一起后不被世人理解。
陈泽遭到霸凌，展虎被家里警告。

两个人最后为了对方都能好过，分开了。
做回普通朋友。

但是彼此心里，一直有一道无法跨越的隔阂。

【达成结局：恨海情天】
```

**CG 画面**：
- 两人背对背站立
- 中间隔着看不见的墙
- 背景是黄昏的街道

---

### 结局二：朋友以上（Normal End）

```
陈泽没有听清最后的名字。
他只是对着电话说：别说了...

第二天，展虎像什么都没发生一样。
陈泽也默契地没有提起。

他们还是朋友。
最好的朋友。
也...只能是朋友。

【达成结局：朋友以上】
```

**CG 画面**：
- 两人并肩走在上学路上
- 手微微靠近，但没有牵起
- 背景是清晨的街道

---

### 结局三：禁忌之恋（True End）

```
电话那头沉默了很久。
然后传来一声轻笑。

展虎："那你现在...想清楚了吗？"

陈泽："嗯，想清楚了。"
陈泽："其实我也喜欢你。"

不在乎世人的眼光。
不在乎家人的反对。
只要能和你在一起，就够了。

【达成结局：禁忌之恋】
```

**CG 画面**：
- 两人相拥在路灯下
- 手机掉落在地
- 背景是星空

---

### 结局四：未说完的话（Hidden End）

```
话没说完，电话挂断了。
或许是醉得不轻，或许是故意挂断。

窗外的路灯亮着。
像极了那年夏天，她和他们表白的那个夜晚。

但现在...
已经回不去了。

【达成结局：未说完的话】
```

**CG 画面**：
