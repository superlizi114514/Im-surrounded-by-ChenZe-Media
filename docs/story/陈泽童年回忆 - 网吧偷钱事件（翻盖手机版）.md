# 陈泽童年回忆：网吧偷钱事件（翻盖手机版）

> 版本：v1.1
> 核心细节：**奶奶的翻盖手机 + 偷钱上网 + 被撞包摔手机**
> 时代背景：2010 年代初期（翻盖机、英雄联盟、QQ 头像）

---

## 一、设定补充

### 时代背景

| 元素 | 设定 |
|------|------|
| 时间 | 2012-2013 年左右 |
| 手机 | 奶奶的旧翻盖机（诺基亚/摩托罗拉） |
| 游戏 | 英雄联盟（S2-S3 赛季） |
| 网吧 | 红蚂蚁网吧（当地最好，2 元/小时） |
| QQ | 非智能机 QQ，头像需要电脑换 |

### 陈泽家庭设定

```
陈泽家庭情况：
- 奶奶：管得特别严，不让玩手机，不让去网吧
- 父母：在外地打工，奶奶负责照顾
- 零花钱：很少，不够上网
- 结果：偷钱去网吧，被撞到后慌乱摔了奶奶的手机
```

## 五、情感增强点

### 优化前
- 摔手机 = 普通慌乱
- 头像 = 普通浪漫

### 优化后
- 摔手机 = **奶奶的遗物般的老手机** + **怕被打死** + **胶布缠着继续用**
- 头像 = **专门去打印店做的图** + **黑色背景白色宋体** + **用裂屏翻盖机也换了**

**情感冲击力提升**：
1. "他那么宝贝那部手机，却为了我换了头像"
2. "手机摔坏了用胶布缠着，但头像一定要换成她的字"
3. "奶奶催他回家的短信，他直接按掉，因为要跟喜欢的人表白"
