## 八、常见问题解决

### 问题：脸部崩坏
```
解决：加`精致脸部，完美脸型`
负向：`崩坏脸，变形脸`
```

### 问题：手部崩坏
```
解决：加`精致手部，完美手部`
或避免手部特写
负向：`崩坏手，缺失手指`
```

### 问题：风格不一致
```
解决：固定同一个模型/ checkpoint
使用相同种子范围
```
