# 素材总汇

## 目录结构

```
assets/
├── in-game/                  # 游戏内使用的素材（已导入游戏目录）
│   ├── figures/              # 角色立绘（按角色分类）
│   ├── backgrounds/          # 背景图片
│   ├── vocals/               # 配音文件（按角色分类）
│   ├── cg/                   # CG 事件图
│   └── items/                # 道具素材（书包、手机、座机等）
│
└── extra/                    # 额外素材（未直接入游戏，可供二次创作）
    ├── raw-images/           # AI 生成原图（未经裁剪/调色）
    ├── chapter-materials/    # 各章节人物素材（按章节分）
    │   ├── prologue/         # 序章人物素材
    │   ├── chapter1/         # 第一章人物素材
    │   ├── chapter2/         # 第二章人物素材
    │   ├── ...               # 后续章节
    │   └── general/          # 通用素材
    ├── three-view/           # 角色三视图模板（正面+侧面）
    │   ├── 陈泽/
    │   ├── 陈志涛（黑白）/
    │   ├── 崔立伟（老崔）/
    │   ├── 展虎/
    │   ├── 刘朔军/
    │   └── 小臭屋泽/
    ├── audio/                # 额外音频文件
    └── raw-photos/           # 参考照片/原图
```

## 统计

| 类别 | 数量/大小 |
|------|-----------|
| 角色立绘 | 111+ 张 |
| 背景图片 | 90+ 张 |
| 配音文件 | 500+ 条（含 flac+mp3 双格式） |
| 过场视频 | 7 个 |
| BGM | 5 首 |
| CG | 数张 |
| 三视图模板 | 6 个角色 |

## 素材来源

- AI 绘画生成（即梦、Banana 等工具生成后筛选）
- 部分素材经 Photoshop 等工具后期处理
- 配音使用 AI 语音合成生成

## 使用建议

- **游戏内素材**：可直接用于 WebGAL 或 Ren'Py 等引擎
- **额外素材**：原图可自行裁剪、调色后使用；三视图适合做角色建模参考
- **配音建议**：部分配音为 AI 生成，可后续找人补录提升质量
