# 角色三视图模板

角色三视图（正面 + 侧面），适用于角色建模参考或 AI 生图一致性控制。

## 角色清单

| 角色 | 文件 |
|------|------|
| **陈泽** | `泽/泽.png` + 正侧面 |
| **陈志涛（黑白）** | `陈志涛/`（3 张） |
| **崔立伟（老崔）** | `崔立伟/`（1 张） |
| **展虎（虎）** | `虎/`（1 张） |
| **刘朔军（军）** | `刘朔军/`（1 张） |
| **小臭屋泽** | `小臭屋泽/`（1 张） |

## 使用方法

1. 参考三视图确定角色外观
2. 使用 AI 生图工具（ComfyUI + ControlNet）生成多角度立绘
3. 或在角色设计时保持外观一致性

## 来源

三视图模板由 AI 生成后手工筛选，供角色设计参考。
