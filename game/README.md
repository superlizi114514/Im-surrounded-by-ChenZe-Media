# 游戏主体 - WebGAL 版

基于 [WebGAL](https://github.com/OpenWebGAL/WebGAL) 引擎构建。

## 目录说明

| 目录/文件 | 说明 |
|-----------|------|
| `config.json` | 游戏配置（分辨率 1920x1080） |
| `scene/` | 剧本文件（`.txt` 格式，WebGAL Script 语法） |
| `figure/` | 角色立绘（111+ 张，含表情差分） |
| `background/` | 场景背景（90+ 张） |
| `audio/` | 音频文件 |
| `bgm/` | 背景音乐（5 首） |
| `vocal/` | 角色配音（100+ 条，按角色分类） |
| `video/` | 过场视频（7 个） |
| `CG/` | 事件 CG |
| `sfx/` | 音效 |
| `gui/` | UI 资源（字体等） |
| `animation/` | 动画配置 |
| `items/` | 道具素材 |

## 运行方式

1. 打开 WebGAL Terre 编辑器，导入此目录
2. 或直接打开 `index.html`（需 WebGAL 引擎支持）
