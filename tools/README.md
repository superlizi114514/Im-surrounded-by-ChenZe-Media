# 工具汇

## 包含工具

| 工具 | 说明 |
|------|------|
| [WebGAL 文本编辑器](https://github.com/OpenWebGAL/WebGAL_Terre) | WebGAL 引擎官方可视化编辑器 |
| `scripts/convert_audio.bat` | 音频格式转换（FLAC → MP3） |
| `scripts/convert_audio.py` | Python 版音频格式转换脚本 |

## 推荐工具清单

### AI 生图
- [即梦](https://jimeng.jianying.com/) - 字节跳动出品，适合角色/背景生成
- [Banana](https://banana.dev/) - AI 生图/视频工具
- [ComfyUI](https://github.com/comfyanonymous/ComfyUI) - 本地 Stable Diffusion 工作流（推荐 B 站 [啦啦啦的小黄瓜](https://space.bilibili.com/219572544) 教程）

### AI 配音
- 各类 TTS 工具（注意：字节跳动的工具目前人脸处理有限制）
- 推荐使用本地 TTS 方案或在线配音平台

### 视频制作
- ComfyUI + AnimateDiff 工作流

### 游戏开发
- [WebGAL](https://github.com/OpenWebGAL/WebGAL) - Web 视觉小说引擎
- [WebGAL Terre](https://github.com/OpenWebGAL/WebGAL_Terre) - WebGAL 可视化编辑器
